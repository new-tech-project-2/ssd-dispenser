# how-to-use
* my_websocket을 실행합니다.
* nfc 모듈에 nfc 태그를 접촉할 시 backend 서버에 정보를 전송합니다.
* socket을 통해 backend 서버에서 전송받은 디스펜서 모드 정보를 업데이트 합니다.
